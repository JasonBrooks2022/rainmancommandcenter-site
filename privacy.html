Privacy Policy

Effective Date: March 16, 2026

Rainman Security Wizard ("we", "our", "us") values your privacy. This policy explains what information the app collects and how it is used.

Information We Collect

  • Account information (Google account email/ID) when you sign in.
  • User data such as browser sync data (bookmarks, history, settings) when sync is enabled.
  • Credentials only when you explicitly save with Android/Google Password Manager.
  • Basic diagnostics for stability and troubleshooting.

How We Use Information

  • Provide sign-in and cloud sync.
  • Save and restore app data across devices.
  • Improve reliability and security.

Google APIs / Drive Access

If you connect Google Drive, access is used only for operations you request (browse/upload/download/manage files).

Data Sharing

We do not sell personal data. Data is shared only with services required for app features (for example Google/Firebase).

Data Retention

Synced data is retained until deleted by you or until your account connection is removed.

Security

We use platform and service security controls, but no method of transmission or storage is 100% secure.

Children's Privacy
